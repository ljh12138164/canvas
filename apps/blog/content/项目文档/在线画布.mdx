# LJH-Design设计系统

## 项目介绍
LJH-Design是一个综合性的设计系统平台，集成了设计资源管理、AI辅助设计等功能，帮助设计师提高工作效率。

## 技术栈
- Next.js - React框架
- TypeScript - 类型系统
- TailwindCSS - 样式框架
- shadcn/ui - UI组件库
- Tanstack Query - 数据请求
- Server Side Rendering - 服务端渲染

## 核心功能
1. 设计系统
   - 组件库
   - 设计规范
   - 资源管理

2. AI助手
   - 智能设计生成
   - 设计优化建议
   - 自动布局

3. 素材中心
   - 素材管理
   - 分类检索
   - 版本控制

4. 数据分析
   - 使用统计
   - 性能监控
   - 用户反馈

## 项目结构
```
apps/ljh-design/
├── app/              # Next.js应用目录
│   ├── components/   # 组件
│   ├── hooks/       # 自定义Hook
│   └── types/       # 类型定义
├── public/          # 静态资源
└── styles/         # 样式文件
```

## 开发指南
1. 安装依赖
```bash
pnpm install
```

2. 启动开发服务器
```bash
pnpm dev
```

3. 构建生产版本
```bash
pnpm build
```

## 部署要求
- Node.js 18+
- SSR支持
- 现代浏览器
