# 谷歌AI图片分析接口说明

## 接口概述

该服务提供了4个主要接口，用于处理图片分析需求：

### 1. 图片URL分析 (`POST /image`)
- 接收图片URL进行分析
- 参数：
  - `image`: 图片URL
  - `prompt`: 可选的提示词
  - `filetype`: 图片类型

### 2. 图片URL流式分析 (`POST /imageStream`) 
- 支持流式响应的图片URL分析
- 参数：
  - `image`: 图片URL
  - `prompt`: 可选的提示词
  - `history`: 可选的对话历史

### 3. 文件上传分析 (`POST /file`)
- 直接上传图片文件进行分析
- 参数：
  - `image`: 图片文件
  - `prompt`: 可选的提示词

### 4. 文件上传流式分析 (`POST /fileStream`)
- 支持流式响应的文件上传分析
- 参数：
  - `image`: 图片文件
  - `prompt`: 可选的提示词
  - `history`: 可选的对话历史

## 特点
- 所有接口默认使用中文回复
- 支持自定义提示词
- 支持对话历史记录
- 提供流式和非流式两种响应方式
- 内置错误处理机制

## 默认配置
- 默认提示词：详细解释图片内容，包括主要元素、场景、特点等
- AI参数配置：
  - temperature: 0.7
  - topK: 40
  - topP: 0.95
