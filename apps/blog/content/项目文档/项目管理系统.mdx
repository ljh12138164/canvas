# Jebet项目管理系统

## 项目介绍
Jebet是一个现代化的团队协作平台，专注于提供高效的项目管理和团队协作解决方案。

## 技术栈
- React - UI框架
- TypeScript - 类型系统
- Vite - 构建工具
- TailwindCSS - 样式框架
- shadcn/react - UI组件库
- Tanstack Query - 数据请求
- MobX - 状态管理
- Socket.io - 实时通信

## 核心功能
1. 工作区管理
   - 多工作区支持
   - 成员管理
   - 权限控制

2. 项目管理
   - 项目创建与配置
   - 任务分配
   - 进度追踪

3. 团队协作
   - 实时消息
   - 文件共享
   - 协作编辑

4. 数据统计
   - 项目进度报告
   - 工作量统计
   - 性能分析

## 项目结构
```
apps/jebet/
├── src/               # 源代码
│   ├── components/    # 组件
│   ├── hooks/        # 自定义Hook
│   ├── store/        # 状态管理
│   └── types/        # 类型定义
├── public/           # 静态资源
└── e2e/             # 端到端测试
```

## 开发指南
1. 安装依赖
```bash
pnpm install
```

2. 启动开发服务器
```bash
pnpm dev
```

3. 运行测试
```bash
pnpm test
```

## 部署要求
- Node.js 18+
- WebSocket支持
- 现代浏览器
